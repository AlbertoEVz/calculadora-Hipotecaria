# Calculadora Hipotecaria

Como parte del curso que tome desarrolle una calculadora hipotecaria y un módulo de registro de usuarios con operaciones CRUD, utilizando HTML, CSS, JavaScript y Bootstrap. Implementé formularios, funciones personalizadas, manipulación del DOM y almacenamiento de datos en Local Storage, mejorando habilidades en diseño responsivo y gestión dinámica de información.

Además, utilice herramientas como w3schools e infinityFree para complementar el proyecto.

Como funciona:

En esta parte el usuario ingresa la cantidad de la cuota inicial con la que cuenta el interesado, seguido del valor del inmueble, la tasa de interés y el plazo en años.

# Registro de usuario

<img width="1251" height="251" alt="image" src="https://github.com/user-attachments/assets/9457f276-186a-46ae-9ab7-e8de7793cb41" />

En la parte de inscripción de usuario es un formulario donde se ingresan datos como: Nombre(s), Apellidos, número de teléfono, e-mail, Ciudad y País.

<img width="1247" height="580" alt="image" src="https://github.com/user-attachments/assets/ce6994bf-2b5b-4491-b64f-40498361d1ec" />

Una vez llenados los campos podemos guardar los datos del usuario.

<img width="921" height="331" alt="image" src="https://github.com/user-attachments/assets/460c0af2-093a-4602-b0c5-330fd5cefa98" />

Si presionamos el botón “listar” nos mostrara el registro de los usuarios y dentro del listado nos permite “Ver” el contacto, “Editar” o “Eliminar”.

<img width="921" height="192" alt="image" src="https://github.com/user-attachments/assets/cc07ce08-881a-475e-99cb-28296bbca049" />

En este punto los datos registrados se almacenan en el local storage del navegador pero si se desea se puede modificar el código para conectarla a alguna base de datos de la preferencia del usuario.
